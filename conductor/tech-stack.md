# Tech Stack

This document outlines the technology stack used in the Agent Pipeline Framework project.

*   **Primary Language:** Scala 3
*   **Core Framework:** ZIO (for building concurrent and asynchronous applications)
*   **Messaging System:** RabbitMQ (for inter-agent communication)
*   **LLM Integration:** Ollama (for local LLM inference)
*   **JSON Serialization:** zio-json (for type-safe JSON encoding/decoding)
*   **Build Tool:** sbt (Scala Build Tool)
*   **Scripting:** Python (for utility scripts and tool execution)
